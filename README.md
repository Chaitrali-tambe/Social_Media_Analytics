## Demonstration of Data analytics
These are some demonstrations data analytics on the data collected from social media or online services like tweets, google reviews, comments etc.

⭐ EDA on Netflix and DisneyPlus data
A comparative Exploratory Data Analysis (EDA) of Netflix and Disney+ Hotstar to uncover contrasting content strategies, audience targeting, and market positioning. This project transforms raw streaming data into actionable business insights using Python.

⭐ Sentiment Analysis on Amazon Data
An end-to-end Natural Language Processing (NLP) project focused on classifying Amazon Customer Reviews. By analyzing text patterns and star ratings, this project identifies consumer sentiment (Positive, Negative, Neutral) to help brands understand product feedback at scale.

⭐ Sentiment Analysis on Turkey Earthquake
This is real-time sentiment analysis and emotion detection on a large-scale dataset of tweets related to the February 2023 Turkey-Syria Earthquake. By utilizing Natural Language Processing (NLP), it investigates the dynamic nature of public responses, emotional reactions, and key information flows during a major natural disaster.

⭐ Social Network analysis - Facebook Food page
This demonstration is to understand how communities are formed and how information spreads. By treating pages as nodes and their mutual likes/interactions as edges, it identifies influential hubs and hidden sub-communities within the culinary digital space.

⭐ Topic Modelling using LDA - Google Reviews for McDonalds
This project utilizes Latent Dirichlet Allocation (LDA) to automatically discover hidden themes across thousands of Google Reviews for McDonald’s. By moving beyond simple star ratings, the analysis extracts the specific "topics" (e.g., service speed, food quality, kiosk issues) that drive customer satisfaction and dissatisfaction.
